# Добавление картинок в админке
- Что нужно сделать
1. Создать папку frontend/public/assets/items/ или другая конечная папка
2. Положить туда картинки в .webp (лучший формат для веба — меньше размер, есть альфа-канал)
3. В админке указывать путь /assets/items/имя_файла.webp

# 2. Создать триггерную статью (через админку или API)
Бэкенд готов, нужно создать статью с триггером. Через POST /admin/articles:
{
  "chapter_id": "<id секретной главы>",
  "title": "Куда все ушли?",
  "content": "Текст скрытой статьи...",
  "trigger_event_type": "stare_60s",
  "trigger_threshold": 1
}
